Projects Completed

### Task 1: Iris Flower Classification
* **Objective:** Train a machine learning model to classify iris flowers into three species based on sepal and petal measurements.
* **Tech Stack:** Python, Pandas, Scikit-Learn (K-Nearest Neighbors).
* **Summary:** Built a classification model achieving 100% accuracy on the test set. 

### Task 2: Unemployment Analysis with Python
* **Objective:** Analyze the sharp increase in India's unemployment rate during the COVID-19 pandemic.
* **Tech Stack:** Python, Pandas, Matplotlib, Seaborn.
* **Summary:** Cleaned the dataset and utilized Exploratory Data Analysis (EDA) to visualize the exact economic impact timeline and identify the hardest-hit states.
